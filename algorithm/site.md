## 位运算

- & 按位与，两者都为 1，才是 1
- | 按位或，一个为 1，就是 1
- ^ 按位异或，只有一个为 1，才是 1
- ~ 按位取反，1 变 0， 0 变 1
- << 左移
-     >> 右移

一般情况下，位运算比符号运算更快，例如

```
a = 7 % 2 = 1, b = 8 / 2 = 4;
a = 7 & 1 = 1, b = 8 >> 1  = 4
x = x & (x - 1) 清零最低位的1

```

c = a ^ b, => a = c ^ b; b = a ^ c; </br>
x ^ x = 0; </br>
x & 1 = 1; </br>
