注：代码执行顺序是：代码输入--->编译阶段--->执行阶段

### 变量提升

变量声明会在编译阶段处理，而变量赋值则会原地等待执行

```
console.log(name); //undefined;
var name = 'lili';
console.log(name); // lili
```

### 函数提升

解析器在解析时，对函数声明和变量声明有着不同的优先级，在编译时，**函数声明会优先于变量声明**

#### 函数声明不会被变量声明覆盖，但会被变量赋值覆盖

```
showName()
var showName = function() {
  console.log(2)
}
function showName() {
  console.log(1)
} //out: 1
```

```
var showName = function() {
  console.log(2)
}
function showName() {
  console.log(1)
}
showName() //out: 2
```
